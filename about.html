About this app

This application was generated automatically using the TEI Processing Model App Generator. Only the logo and background images have been uploaded afterwards. Apart from this, no code has been added or modified.

The primary purpose of this app is to test the TEI Processing Model on a larger corpus of texts. Instead of using an XSLT stylesheet or XQuery, all texts are rendered by interpreting processing model instructions described in an ODD file. This saves us many lines of code (more than 2000 lines of XQuery in this example). All output formats are defined by the same ODD!

Not much attention has been paid to search yet. It more or less uses the out of the box search features of eXistdb.
